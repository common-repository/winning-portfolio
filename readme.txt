== Installation ==

Winning Portfolio can be installed in two different ways:

*  Via WordpPress uploader
*  or via FTP Client

1- Steps to follow to install plugin via WordPress uploader

*   go to WordPress admin panel and click on "plugins" from the dashboard menu.
*   Then from Plugins -> Add New -> Upload -> Choose file -> and then choose winning-portfolio.zip and press Open-> Install Now -> Activate Plugin

2- Steps to follow to install plugin via FTP

*   Unzip the file
*   Upload unziped winning-portfolio folder to your WordPress plugins directory (/wp-content/plugins/)
*   Activate from WordPress menu Plugins section
